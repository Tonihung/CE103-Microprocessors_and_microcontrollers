# Laboratory Exercises

This directory contains the laboratory exercises for the **Microprocessors and Microcontrollers** course.

Each laboratory folder contains:

- `code/` - Assembly source code extracted from the laboratory report
- `pdf/` - Laboratory report
- `README.md` - Short description of the exercise

## Laboratory List

- [LAB01](./LAB01) - Proteus and the 8051 Microcontroller Family
- [LAB02](./LAB02) - Communication with 7-Segment LED and Timer
- [LAB03](./LAB03) - Using Interrupts
- [LAB04](./LAB04) - Using UART
- [LAB05](./LAB05) - Addition and Subtraction of Two 32-bit Numbers on the 8086 Processor
- [LAB06](./LAB06) - I/O Processing, Calculation, and Memory on the 8086 Processor
