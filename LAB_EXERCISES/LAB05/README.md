# LAB05 - 32-bit Addition and Subtraction on the 8086 Processor

This laboratory performs 32-bit addition and subtraction on the **8086 processor** using pairs of 16-bit words, carry/borrow propagation, and signed-overflow detection.

- [Assembly source code](./code/add_sub_32bit.asm)
- [Laboratory report](./pdf/LAB05_Report.pdf)

The default operands in the source file use the first example from the report and can be edited through `x_hi`, `x_lo`, `y_hi`, and `y_lo`.
